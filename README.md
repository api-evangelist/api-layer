# API Layer (api-layer)
APILayer is a curated API marketplace offering 178+ production-ready APIs across categories including finance, currency exchange, geolocation, weather, AI/ML, security, and developer tools, with API key authentication, free tiers, and comprehensive documentation. It hosts well-known APIs including IPstack, Currencylayer, Fixer, Weatherstack, Aviationstack, and Mediastack.
